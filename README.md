# Spotify Clone – HTML & CSS

This project is a UI clone of the Spotify Web Player (Home Page) built using only HTML and CSS.

## Features
- Sidebar navigation
- Recently Played section
- Cards-based layout
- Clean Spotify-like UI

## Tech Stack
- HTML5
- CSS3
